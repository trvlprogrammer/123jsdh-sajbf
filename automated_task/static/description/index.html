Create Task Automatically

Overview

Sometimes we have routine task like checking email everyday, maintenance server every month, monitoring server, and etc. When the schedule of the routine comes, you have to make the same task again and it's wasting time. Automated task will do it for you, you don't have to waste your time to make the same task again and again, what you need is just 1 simple configuration to do it.

Automated Configuration

1. Go to Project menu, bellow Project menu you will find Automated Tasks menu.


2. Create new automated task and fill the form. You can set when you want to create task automatically, automated task has 4 types. Day means you can create the task every day, if you choose day you will see additional field called daily, you can choose the number, 1 for every day, 2 for every 2 days and so on. Weekdays means the task will be created every monday to fiday. Week, you can define what day the task will be created. Month, task will be created every month(30 days).

Repetition

Automated task has 4 repetition types, day, weekdays, week and month.




Task created by automated task

You can see list of task that created by automated task. In form view you can see Tasks tab next to Description tab, in there you can see task that created by automated task.

Schedule

If you want to see the schedule when the next task will be created just go to debug mode and tab Create on will be appeared next to tab Tasks.
