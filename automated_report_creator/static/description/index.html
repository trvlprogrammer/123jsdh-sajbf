Scheduler Report Creator

Overview

Sometimes we need to generate report in a period time, like every an hour, 6 hours or 1 week. With this module we don't have to do that thing manually again, it will be done by system and we can choose when we want to generate the report.

Execute Query Configuration

1. Open report definition in form view.
2. Check for automatically generated and choose the time when you want to generate the report
very simple right?
