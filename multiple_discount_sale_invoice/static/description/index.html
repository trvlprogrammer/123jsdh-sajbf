Multiple Discounts For Sale & Invoicing

Overview

Sometimes we need to add multiple discount in our sales. This module will allow you to add multiple discount by click the discount amount that you want. You can select discount based on discount type, Fixed Amount or Percentage.

Discount Configuration

Only user with access sales manager who can make discount configuration.
1. Go to Sales menu, bellow Configuration you will find Discount Configuration menu.
2. Create new discount configuration, create discount based on discount type (Fixed Amount / Percentage).
3. Multiple Discount ready to use.

Applying Multiple Discount On Quotations

1. Go to Sale menu and create new Quotations.
2. Select Discount type, then select Discount amount.


Applying Multiple Discount On Customer Invoices

1. Go to Invoicing menu and create new invoices.
2. Select Discount type, then select Discount amount.
