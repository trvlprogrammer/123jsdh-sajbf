Execute Query From Odoo Application & Running Automatically

Overview

Sometimes we need to run quick query in our application to check or update a data, but we have to do several process like entering into server and open our pgAdmin. With this module you don't have to do that anymore, you can execute query from your application now.

Execute Query Configuration

1. Enter to debug mode.
2. Go to Settings -> Technical -> Execute Query




3.Define your query, if you want to test your query, you can run it manually, just click the button run manually.


Features

1.Running by scheduler, you can choose when this query will be executed.


2.Has 3 type query, select, update, delete.
If you choose select you can see the result of your query, if you choose update/delete you can see the return of the query success or not.
